What Is a Variable?

Imagine you have a piggy bank. You can put coins inside it, take coins out, or add more whenever you want.

A variable in programming works the same way. It is a place where we store a value, just like a piggy bank stores coins!

Changing What’s Inside

Let’s say your piggy bank starts with 0 coins.

You decide to put 5 coins inside. Now the value is 5. Later, you add 1 more coin, and the value becomes 6.

Obeserve how the number can change. That’s what programmers do with variables! They can update their value any time.
